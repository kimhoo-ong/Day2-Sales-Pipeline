# Day2-Sales-Pipeline

This PBIP project has been adapted for the `day02_sales_pipeline.csv` dataset.

Local source file used by the semantic model:

- `data/day02_sales_pipeline.csv`

Current project setup:

- `FactPipeline` import table with derived quality and reporting columns
- Pipeline KPI measures for forecasting and funnel health
- Four blank report pages ready for visual layout:
  - Executive Overview
  - Sales Rep Performance
  - Source and Industry
  - Deal Risk and Action

Open `Blank_file.pbip` in Power BI Desktop and refresh the model. After that, you can place visuals using the measures and page plan from the challenge notes.
